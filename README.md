# Obesity Level Prediction Classification

## Overview

This project aims to classify obesity levels based on eating habits, physical activity, and personal characteristics using machine learning techniques.

## Dataset

The dataset contains information about eating habits, physical condition, and lifestyle factors that influence obesity levels.

Target variable:

- Insufficient Weight
- Normal Weight
- Overweight Level I
- Overweight Level II
- Obesity Type I
- Obesity Type II
- Obesity Type III

## Machine Learning Algorithms

- Decision Tree
- Random Forest
- XGBoost
